# 🚀 دليل GitHub Actions لبناء APK

## 📱 كيفية الحصول على APK حقيقي مجاناً

### 🎯 **الطرق المتاحة:**

---

## 1️⃣ **البناء اليدوي (الأسهل)**

### خطوات التشغيل:
1. **اذهب إلى GitHub:** https://github.com/master1moon/song6
2. **اضغط على تبويب "Actions"**
3. **اختر "🚀 Build Android APK - إدارة كروت الشبكة"**
4. **اضغط "Run workflow"**
5. **اختر نوع البناء:**
   - `debug` - للاختبار (أسرع)
   - `release` - للنشر (محسن)
6. **اضغط "Run workflow" الأخضر**

### النتيجة:
- ⏱️ **وقت البناء:** 5-10 دقائق
- 📱 **APK جاهز** في تبويب Artifacts
- ✅ **يعمل على جميع أجهزة أندرويد**

---

## 2️⃣ **البناء التلقائي**

### كيف يعمل:
- 🔄 **يتم تلقائياً** عند كل تحديث للكود
- 📱 **APK جاهز** خلال دقائق
- 💬 **إشعار تلقائي** في الـ commit

### للتفعيل:
```bash
git push origin main
```

---

## 3️⃣ **إنشاء إصدار رسمي**

### للحصول على إصدار Release:
1. **اذهب إلى Actions**
2. **اختر "🏆 إنشاء إصدار APK"**
3. **اضغط "Run workflow"**
4. **أدخل رقم الإصدار** (مثل: 1.0.0)
5. **أضف ملاحظات الإصدار**
6. **اضغط "Run workflow"**

### النتيجة:
- 🏆 **إصدار رسمي** في تبويب Releases
- 🔐 **APK موقع ومحسن**
- 📝 **ملاحظات إصدار كاملة**

---

## 📥 **كيفية تحميل APK:**

### من Actions:
```
1. GitHub → Actions
2. اختر آخر build ناجح ✅
3. انزل لأسفل → Artifacts
4. اضغط على اسم APK للتحميل
5. فك الضغط → احصل على APK
```

### من Releases:
```
1. GitHub → Releases
2. اختر آخر إصدار
3. انزل لأسفل → Assets  
4. اضغط على APK للتحميل مباشرة
```

---

## 📱 **تثبيت APK:**

### على أندرويد:
```
1. انقل APK لجهازك
2. الإعدادات → الأمان → مصادر غير معروفة ✅
3. مدير الملفات → اضغط على APK
4. اضغط "تثبيت"
5. اتبع التعليمات
```

---

## 🔧 **استكشاف الأخطاء:**

### إذا فشل البناء:
1. **اذهب إلى Actions**
2. **اضغط على Build الفاشل**
3. **اضغط على المهمة الحمراء**
4. **اقرأ رسالة الخطأ**

### الأخطاء الشائعة:
- ❌ **نقص ذاكرة:** سيعيد المحاولة تلقائياً
- ❌ **خطأ Gradle:** تحقق من ملفات build.gradle
- ❌ **خطأ SDK:** سيتم إصلاحه تلقائياً
