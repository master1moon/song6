# 📸 دليل التقاط لقطات الشاشة للتطبيق

## 🎯 الأقسام المهمة لالتقاط الصور

### 1. 🏠 لوحة التحكم الرئيسية (Dashboard)
- **المسار**: الصفحة الرئيسية عند فتح التطبيق
- **ما يجب إظهاره**:
  - الإحصائيات الرئيسية (عدد الباقات، المحلات، إجمالي الكروت)
  - أرقام المبيعات والمدفوعات والديون
  - صافي الربح مع الألوان التمييزية
- **نصائح التصوير**: تأكد من وجود بيانات تجريبية لإظهار الأرقام

### 2. 📦 إدارة الباقات (Packages)
- **المسار**: النقر على "الباقات" في القائمة الجانبية
- **ما يجب إظهاره**:
  - جدول الباقات مع الأسعار الثلاثة
  - نموذج إضافة باقة جديدة
  - أزرار التحكم (تعديل، حذف)
- **لقطات مطلوبة**: 
  - عرض الجدول الكامل
  - نموذج الإضافة/التعديل

### 3. 🏪 إدارة المحلات (Stores)
- **المسار**: النقر على "المحلات" في القائمة الجانبية
- **ما يجب إظهاره**:
  - قائمة المحلات مع الأرصدة
  - تفاصيل محل واحد مع سجل المعاملات
  - نموذج إضافة محل جديد
- **لقطات مطلوبة**:
  - القائمة الجانبية للمحلات
  - صفحة تفاصيل محل
  - نموذج الإضافة

### 4. 📊 إدارة المخزون (Inventory)
- **المسار**: النقر على "المخزون" في القائمة الجانبية
- **ما يجب إظهاره**:
  - جدول المخزون مع الكميات
  - تحذيرات المخزون المنخفض
  - نموذج إضافة/تعديل المخزون
- **لقطات مطلوبة**:
  - عرض الجدول
  - التحذيرات
  - النموذج

### 5. 💰 إدارة المبيعات (Sales)
- **المسار**: النقر على محل معين ثم "إضافة بيع"
- **ما يجب إظهاره**:
  - نموذج إضافة بيع (باقة أو مخصص)
  - سجل المبيعات لمحل
  - حساب الإجماليات التلقائي
- **لقطات مطلوبة**:
  - نموذج البيع العادي
  - نموذج البيع المخصص
  - سجل المبيعات

### 6. 💳 إدارة المدفوعات (Payments)
- **المسار**: النقر على محل ثم "إضافة دفعة"
- **ما يجب إظهاره**:
  - نموذج إضافة دفعة
  - سجل المدفوعات
  - تأثير الدفعة على الرصيد
- **لقطات مطلوبة**:
  - نموذج الدفعة
  - سجل المدفوعات

### 7. 💸 إدارة المصروفات (Expenses)
- **المسار**: النقر على "المصروفات" في القائمة
- **ما يجب إظهاره**:
  - جدول المصروفات مع التصنيفات
  - نموذج إضافة مصروف
  - ميزات البحث والفلترة
- **لقطات مطلوبة**:
  - الجدول الرئيسي
  - نموذج الإضافة
  - خيارات الفلترة

### 8. 📈 التقارير (Reports)
- **المسار**: النقر على "التقارير" في القائمة
- **ما يجب إظهاره**:
  - تقرير الأرباح والخسائر
  - تقرير الديون
  - خيارات التصدير
- **لقطات مطلوبة**:
  - كل نوع تقرير
  - خيارات التصدير

### 9. ⚙️ الإعدادات (Settings)
- **المسار**: النقر على "الإعدادات"
- **ما يجب إظهاره**:
  - إعدادات GitHub
  - خيارات النسخ الاحتياطي
  - الإعدادات العامة
- **لقطات مطلوبة**:
  - جميع تبويبات الإعدادات

### 10. ☁️ المزامنة السحابية
- **المسار**: في الإعدادات → GitHub Sync
- **ما يجب إظهاره**:
  - نموذج إعداد GitHub
  - خيارات الرفع والتحميل
  - رسائل النجاح/الفشل
- **لقطات مطلوبة**:
  - نموذج الإعداد
  - عمليات المزامنة

## 🎨 نصائح للتصوير الاحترافي

### إعداد البيانات التجريبية
1. أضف 5-10 باقات مختلفة
2. أضف 10-15 محل بأرصدة متنوعة
3. أضف مخزون للباقات
4. سجل مبيعات ومدفوعات
5. أضف مصروفات متنوعة

### إعدادات التصوير
- **دقة الشاشة**: 1920x1080 أو أعلى
- **مستوى التكبير**: 100% في المتصفح
- **المتصفح المفضل**: Chrome أو Firefox
- **وضع المطور**: F12 لضبط الأبعاد

### تحسين الصور
- اقتطع الأجزاء غير المهمة
- أضف إطارات أو ظلال خفيفة
- استخدم أدوات مثل Canva لتحسين الصور
- أضف عناوين أو شروحات إذا لزم الأمر

## 📱 لقطات شاشة للأجهزة المختلفة

### سطح المكتب (Desktop)
- الشاشة الكاملة مع القائمة الجانبية
- التركيز على تفاصيل الجداول
- إظهار الميزات التفاعلية

### الأجهزة اللوحية (Tablet)
- واجهة متجاوبة
- القوائم المنسدلة
- تجربة اللمس

### الهاتف المحمول (Mobile)
- القائمة الجانبية المطوية
- النماذج المحسنة للهاتف
- الأزرار المناسبة للمس

## 🎯 استخدام الصور في التسويق

### للمنشورات
- اجمع 3-4 صور في صورة واحدة
- أضف نصوص توضيحية
- استخدم ألوان جذابة

### للموقع أو المتجر
- صور عالية الجودة
- تسلسل منطقي للميزات
- إظهار سهولة الاستخدام

### للفيديو التوضيحي
- خطط لتسلسل الشاشات
- اجعل الانتقالات سلسة
- أضف تعليق صوتي إذا أمكن
