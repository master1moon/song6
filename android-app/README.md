# 📱 تطبيق إدارة كروت الشبكات اللاسلكية - الأندرويد

## 📋 نظرة عامة

تطبيق أندرويد أصلي متكامل لإدارة مبيعات كروت الشبكات اللاسلكية، مصمم خصيصاً للسوق العربي باستخدام أحدث تقنيات تطوير الأندرويد.

## ✨ الميزات الرئيسية

### 🏪 إدارة المحلات والعملاء
- قاعدة بيانات شاملة للمحلات
- ثلاثة أنواع أسعار (تجزئة، جملة، موزعين)
- حساب الأرصدة التلقائي
- اتصال مباشر بالعملاء
- تكامل مع الخرائط للمواقع

- إدارة شاملة للباقات مع أسعار متعددة

- تنبيهات المخزون المنخفض
- تقارير مفصلة للمخزون

### 💰 إدارة المبيعات والمدفوعات

### 📊 التقارير والتحليلات
- رسوم بيانية تفاعلية
- تقارير الأرباح والخسائر

### 🔔 الإشعارات الذكية
- تنبيهات المخزون المنخفض
- تذكير المدفوعات المتأخرة

## 🛠️ التقنيات المستخدمة

### Core Technologies
- **Kotlin** - لغة البرمجة الحديثة
- **Android SDK 34** - أحدث إصدار
- **Material Design 3** - تصميم Google الحديث
- **Jetpack Components** - مكونات أندرويد المتقدمة

### Architecture
- **MVVM Pattern** - نمط معماري حديث
- **Repository Pattern** - فصل طبقات البيانات
- **Dependency Injection (Hilt)** - حقن التبعيات
- **LiveData & Flow** - إدارة البيانات التفاعلية
- **Coroutines** - البرمجة غير المتزامنة

### Database
- **Room Database** - قاعدة بيانات محلية متقدمة
- **SQLite** - قاعدة بيانات محسنة
- **Type Converters** - تحويل البيانات المعقدة
- **Foreign Keys** - العلاقات بين الجداول
- **Indices** - فهرسة للبحث السريع

### UI/UX
- **View Binding** - ربط آمن للعروض
- **RecyclerView** - قوائم محسنة
- **ViewPager2** - تبويبات سلسة
- **Material Components** - مكونات Material
- **Custom Animations** - تأثيرات حركية مخصصة

### Features
- **Barcode Scanner (ZXing)** - مسح الباركود
- **Charts (MPAndroidChart)** - رسوم بيانية
- **PDF Generation (iText)** - إنشاء PDF
- **Excel Export (Apache POI)** - تصدير Excel
- **Image Loading (Glide)** - تحميل الصور
- **Permissions (EasyPermissions)** - إدارة الصلاحيات

## 🚀 كيفية تشغيل التطبيق

### المتطلبات
- Android Studio Hedgehog (2023.1.1) أو أحدث
- Android SDK 34
- JDK 17 أو أحدث
- جهاز أندرويد أو محاكي (API 24+)

### خطوات التشغيل

1. **فتح المشروع:**
   ```bash
   # فتح مجلد android-app في Android Studio
   File → Open → اختيار مجلد android-app
   ```

2. **مزامنة المشروع:**
   ```bash
   # في Android Studio
   Tools → Sync Project with Gradle Files
   ```

3. **بناء المشروع:**
   ```bash
   # من Terminal في Android Studio
   ./gradlew build
   ```

4. **تشغيل التطبيق:**
   ```bash
   # تثبيت على الجهاز/المحاكي
   ./gradlew installDebug
   
   # أو الضغط على زر Run في Android Studio
   ```

## 📦 البناء للإنتاج

### إنشاء APK للاختبار
```bash
./gradlew assembleDebug
```

### إنشاء Bundle للنشر
```bash
./gradlew bundleRelease
```

### توقيع التطبيق
1. إنشاء Keystore:
   ```bash
   keytool -genkey -v -keystore network-cards-key.keystore -alias network-cards -keyalg RSA -keysize 2048 -validity 10000
   ```

2. إضافة معلومات التوقيع في `app/build.gradle`:
   ```gradle
   android {
       signingConfigs {
           release {
               storeFile file('network-cards-key.keystore')
               storePassword 'your-password'
               keyAlias 'network-cards'
               keyPassword 'your-password'
           }
       }
   }
   ```

## 🏪 النشر في Google Play Store

### متطلبات النشر
1. ✅ **حساب مطور Google Play** ($25 رسوم لمرة واحدة)
2. ✅ **تطبيق موقع ومختبر**
3. ✅ **وصف التطبيق باللغتين العربية والإنجليزية**
4. ✅ **لقطات شاشة عالية الجودة** (8 لقطات على الأقل)
5. ✅ **أيقونة التطبيق** بدقة عالية (512x512)
6. ✅ **سياسة الخصوصية** منشورة على موقع

### خطوات النشر
1. **تحضير الملفات:**
   - APK أو Bundle موقع
   - لقطات الشاشة
   - وصف التطبيق
   - أيقونة عالية الدقة

2. **رفع في Google Play Console:**
   - إنشاء تطبيق جديد
   - رفع الـ Bundle
   - إضافة المعلومات والوصف
   - إعداد التسعير والتوزيع

3. **المراجعة والنشر:**
   - مراجعة Google (1-3 أيام)
   - النشر التدريجي
   - مراقبة التحليلات
