## 📊 الأمثلة العملية للاستخدام

### 🌙 Dark/Light Mode:
```javascript
// تبديل الثيم
toggleTheme();

// تطبيق ثيم محدد
setTheme('dark');

// الحصول على الثيم الحالي
getCurrentTheme();
```

### 📱 Responsive Design:
```javascript
// طي/فتح الشريط الجانبي
toggleSidebar();

// الحصول على معلومات الاستجابة
getResponsiveInfo();

// تطبيق تخطيط مخصص
applyCustomLayout('compact');
```

### ✨ Animations:
```javascript
// تحريك عنصر
animate(element, 'fade-in', { duration: 500 });

// تحريك متدرج
staggeredAnimation(elements, 'slide-in-up', 100);

// تحريك العدادات
animateCounter(element, 1000, 2000);
```

### 🎨 Material Design:
```javascript
// إنشاء زر عائم
createFAB({ icon: 'fas fa-plus', onClick: addNewItem });

// إظهار Snackbar
showSnackbar('تم الحفظ بنجاح!', 'تراجع', undoAction);

// إنشاء كرت Material
createMaterialCard(container, { title: 'عنوان', content: 'محتوى' });
```

### ⏳ Loading States:
```javascript
// تحميل عام
showGlobalLoading('جاري المعالجة...');

// تحميل محلي
const loadingId = showLoading('tableContainer', { type: 'skeleton' });

// تحميل للزر
showButtonLoading(button, 'جاري الحفظ...');

// تحميل مع شريط تقدم
showLoading('container', { type: 'progress', progress: 50 });
```
